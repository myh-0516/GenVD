# ReVeal Dataset

The **ReVeal** dataset is used for source code vulnerability detection and related tasks.  
The original dataset is provided on Hugging Face.

## Source

- Hugging Face Dataset Repository: https://huggingface.co/datasets/claudios/ReVeal

## Download

It is recommended to download the dataset using the **Hugging Face Hub** CLI:

```bash
hf download claudios/ReVeal --repo-type dataset --local-dir ./reveal
